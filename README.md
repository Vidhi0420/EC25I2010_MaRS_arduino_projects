# EC25I2010_MaRS_arduino_projects
This repository includes arduino circuits(along with their working) and mini projects including sensor-based systems and embedded programming concepts built using Tinkercad.

PROJECTS INCLUDED:
- Q1: Blinking LED with different time intervals  
- Q2: RGB LED control using potentiometer  
- Q3: Reaction Time Tester  
- Smart Distance Alert System
- Fire Detection and Alarm System

CONCEPTS LEARNT:
- Digital and analog input/output  
- PWM (Pulse Width Modulation)  
- Timing using millis()  
- Sensor-based systems  
- Embedded programming logic  

DEMONSTRATION:
Each project folder contains:
- Arduino code  
- Circuit design  
- Output video
- A README file that explains about the question/project
